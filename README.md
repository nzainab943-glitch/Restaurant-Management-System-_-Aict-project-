##Project Overview:
This is a menu-driven C program that allows users to place orders for dine-in or home delivery. It handles menu selection, quantity input, and automated billing.

##Key Features:
Multiple food categories (Burgers, BBQ, Chinese, etc.).Includes 8 different food categories and a wide range of drinks and desserts. Automatic discount calculation (up to 25%) and tax application. Dynamic delivery charges based on bill amount. Order type flexibility like Supports both Dine-In and Online Order (Home Delivery) options.
It Collects customer name, phone, and address for delivery orders.

##Calculation Logic:
Discounts: 10% for bills over Rs 3000, 20% for Rs 5000, and 25% for Rs 8000.
Tax: A flat 5% tax is applied to all orders.
Delivery: Charges are scaled (Rs 100, Rs 50, or Free) based on the total bill.
